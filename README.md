# 📊 Análisis de Evasión de Clientes (Churn) – Telecom X

Este proyecto tiene como objetivo analizar la evasión de clientes (*Churn*) en la empresa
Telecom X, utilizando técnicas de análisis exploratorio de datos (EDA) para identificar
patrones y factores asociados a la cancelación del servicio.

El análisis busca transformar datos en información útil que permita apoyar la toma de
decisiones estratégicas orientadas a mejorar la retención de clientes.
